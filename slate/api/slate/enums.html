Core Enumerations

Layout

enum class slate::Layout

Layout: ColMajor, RowMajor.

Values:

Op (Operation)

enum class slate::Op

Operation: NoTrans, Trans, ConjTrans.

Values:

Uplo (Upper/Lower)

enum class slate::Uplo

Uplo: Upper, Lower, General.

Values:

Diag (Diagonal)

enum class slate::Diag

Diagonal: NonUnit, Unit.

Values:

Side

enum class slate::Side

Side: Left, Right.

Values:

Norm

enum class slate::Norm

Norm: One, Two, Inf, Fro, Max, RealOne, RealInf, RealMax.

Values:

Direction

enum class slate::Direction

Direction: Forward, Backward.

Values:

Target

enum class Target : char

Location and method of computation.

Values:

enumerator Host = 'H'

data resides on host

enumerator HostTask = 'T'

computation using OpenMP nested tasks on host

enumerator HostNest = 'N'

computation using OpenMP nested parallel for loops on host

enumerator HostBatch = 'B'

computation using batch BLAS on host (Intel MKL)

enumerator Devices = 'D'

computation using batch BLAS on devices (cuBLAS)

Grid Order

enum class GridOrder : char

Order to map MPI processes to tile grid.

Values:

enumerator Col = 'C'

Column major.

enumerator Row = 'R'

Row major.

enumerator Unknown = 'U'

Unknown (e.g., if using lambda functions).

Options

using slate::Options = std::map<Option, OptionValue>
enum class Option : char

Keys for options to pass to SLATE routines.

Values:

enumerator ChunkSize

chunk size, >= 1

enumerator Lookahead

lookahead depth, >= 0

enumerator BlockSize

block size, >= 1

enumerator InnerBlocking

inner blocking size, >= 1

enumerator MaxPanelThreads

max number of threads for panel, >= 1

enumerator Tolerance

tolerance for iterative methods, default epsilon

enumerator Target

computation method (

See also

Target)

enumerator HoldLocalWorkspace

do not erase local workspace tiles for enabling resue of the tiles by the next routine

enumerator Depth

depth for the RBT solver

enumerator MaxIterations

maximum iteration count

enumerator UseFallbackSolver

whether to fallback to a robust solver if iterations do not converge

enumerator PivotThreshold

threshold for pivoting, >= 0, <= 1

enumerator PrintVerbose = 50

verbose, 0: no printing, verbose, 1: print metadata only (dimensions, uplo, etc.) verbose, 2: print first & last PrintEdgeItems rows & cols from the four corner tiles verbose, 3: print 4 corner elements of every tile verbose, 4: print full matrix

enumerator PrintEdgeItems

edgeitems: number of first & last rows & cols of matrix to print

enumerator PrintWidth

width print format specifier

enumerator PrintPrecision

precision print format specifier For correct printing, PrintWidth = PrintPrecision + 6.

enumerator MethodCholQR = 60

Select the algorithm to compute A^H A.

enumerator MethodEig

Select the algorithm to compute eigenpairs of tridiagonal matrix.

enumerator MethodGels

Select the gels algorithm.

enumerator MethodGemm

Select the gemm algorithm.

enumerator MethodHemm

Select the hemm algorithm.

enumerator MethodLU

Select the LU (getrf) algorithm.

enumerator MethodTrsm

Select the trsm algorithm.

enumerator MethodSVD

Select the algorithm to compute singular values of bidiagonal matrix.

Method Enumerations

enum class slate::MethodCholQR : char

Algorithm to use for Cholesky QR.

Values:

enumerator Auto = '*'

Let SLATE decide.

enumerator GemmA = 'A'

Use gemm-A algorithm to compute A^H A.

enumerator GemmC = 'C'

Use gemm-C algorithm to compute A^H A.

enumerator HerkA = 'R'

Use herk-A algorithm to compute A^H A; not yet implemented.

enumerator HerkC = 'K'

Use herk-C algorithm to compute A^H A.

enum class slate::MethodGels : char

Algorithm to use for least squares (gels).

Values:

enumerator Auto = '*'

Let SLATE decide.

enumerator QR = 'Q'

Use Householder QR factorization.

enumerator CholQR = 'C'

Use Cholesky QR factorization; use when A is well-conditioned.

enum class slate::MethodGemm : char

Algorithm to use for general matrix multiply (gemm).

Values:

enumerator Auto = '*'

Let SLATE decide.

enumerator A = 'A'

Matrix A is stationary, C is sent; use when C is small.

enumerator C = 'C'

Matrix C is stationary, A is sent; use when C is large.

enum class slate::MethodHemm : char

Algorithm to use for Hermitian matrix multiply (hemm).

Values:

enumerator Auto = '*'

Let SLATE decide.

enumerator A = 'A'

Matrix A is stationary, C is sent; use when C is small.

enumerator C = 'C'

Matrix C is stationary, A is sent; use when C is large.

enum class slate::MethodLU : char

Algorithm to use for LU factorization and solve.

Values:

enumerator Auto = '*'

Let SLATE decide.

enumerator PartialPiv = 'P'

Use classical partial pivoting.

enumerator CALU = 'C'

Use Communication Avoiding LU (CALU).

enumerator NoPiv = 'N'

Use no-pivoting LU.

enumerator RBT = 'R'

Use Random Butterfly Transform (RBT).

enumerator BEAM = 'B'

Use BEAM LU factorization.

enum class slate::MethodTrsm : char

Algorithm to use for triangular solve (trsm).

Values:

enumerator Auto = '*'

Let SLATE decide.

enumerator A = 'A'

Matrix A is stationary, B is sent; use when B is small.

enumerator B = 'B'

Matrix B is stationary, A is sent; use when B is large.
